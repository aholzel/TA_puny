# TA_puny

## Author
   Arnold

## About
   This custom command uses the python idna module to encode and decode "punycode" domains

## Usage
   <search> | punydecode <field>
   <search> | punydecode <field> <field2> <mv_field>

   <search> | punyencode <field>
   <search> | punyencode <field> <field2> <mv_field>

   Output will include <field>_decoded with your decoded punycode addresses, or the <field>_encoded with the encoded punycode field.
